# Virtual class for delegating GInteractions

Uses a delegate \`GInteractions\` object during initialization to assign
its \`GInteractions\` slots.

## Value

DelegatingGInteractions virtual class

## Slots

- `delegate`:

  A \`GInteractions\` object used to initialize
  \`GInteractions\`-specific slots.
